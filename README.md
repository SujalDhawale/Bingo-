# 🚀 Multiplayer Bingo (Zero-Config Edition)

This version of Bingo uses **Gun.js**, a decentralized peer-to-peer database. 

## ✨ Why this is better:
- **Zero Configuration**: You don't need Firebase, Ably, or any account. It just works!
- **Serverless**: Works perfectly on Netlify, GitHub Pages, Vercel, and local machines.
- **Instant Sync**: Game state and Chat are synchronized across any device via public relay peers.

## 🚢 How to use:
1.  **Open the link**: [http://localhost:3005](http://localhost:3005) (or your uploaded Netlify/Vercel link).
2.  **Create a Game**: Click "Create New Game".
3.  **Share**: Copy the URL and send it to your friend.
4.  **Play & Chat**: The board and chat will sync instantly!

Everything is set up and ready to go. Enjoy!
